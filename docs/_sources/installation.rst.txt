Installation
============

Dependencies
------------
- Python 3.8+
- numpy
- scipy
- pycbc
- lalsuite (lal)
- astropy

Install dependencies (example using pip):

.. code-block:: bash

   pip install numpy scipy astropy pycbc
   pip install lalsuite

For detailed instructions on installing lalsuite and pycbc, see:
- https://pycbc.org/pycbc/latest/html/install.html
- https://lscsoft.docs.ligo.org/lalsuite/lalsuite/installation.html

If you use conda or HPC modules, adjust accordingly.
